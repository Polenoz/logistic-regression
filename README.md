# Logistic Regression Project

In this project, a logistic regression model is built using Python and scikit-learn. The goal is to classify data based on the Advertising dataset and evaluate how well the model performs.

## Dataset

The project uses the file:

* Advertising.csv

This file should be in the same folder as the notebook, otherwise the code will not run correctly.

## How to run

The easiest way is to use Google Colab:

* Open the notebook in Colab
* Upload the file "Advertising.csv"
* Run all cells from top to bottom

You can also run it locally:

* Install the required libraries (pandas, numpy, matplotlib, seaborn, scikit-learn)
* Open the notebook with Jupyter
* Run all cells

## What is done in this notebook

* Libraries are imported
* The dataset is loaded and inspected
* Basic visualizations are created
* The data is prepared for modeling
* The dataset is split into training and test sets
* A logistic regression model is trained
* Predictions are made
* The results are evaluated

## Evaluation

The model is evaluated using:

* Classification report
* Accuracy score
* Confusion matrix

These metrics show how well the model performs on the test data.
